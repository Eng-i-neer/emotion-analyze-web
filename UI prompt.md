请严格复刻一种「手帐日记 + 治愈系情绪记录网站」UI 风格，整体感觉像打开一本真实的方格手帐本，温柔、松弛、纸质感强，不要做成普通后台管理系统。

## 1. 整体风格关键词

风格定位：
- 手帐风 / 日记本风 / 治愈系 / 温柔陪伴感
- 纸张质感、手绘插画、轻复古、低饱和
- 像学习手帐、情绪日记、心理记录本，而不是冷冰冰的数据面板
- 整体要有“真实桌面 + 翻开的笔记本 + 贴纸 + 便签”的感觉

视觉关键词：
- 米白色纸张
- 方格纸背景
- 手写字体
- 柔和阴影
- 贴纸装饰
- 绿色主色
- 暖黄色辅助色
- 圆角卡片
- 情绪表情图标
- 手绘叶子、太阳、爱心、小植物、咖啡、书本等元素

## 2. 页面整体布局

页面是三栏结构：

### 左侧：固定侧边栏
- 宽度约 220px - 260px
- 背景是米白色纸张质感
- 左上角是产品 Logo：`MindNote ✨`
- Logo 下方有一句小标语：`记录此刻，理解自己。`
- 侧边栏中间是导航菜单
- 菜单项使用图标 + 中文文字
- 当前选中项使用浅绿色背景高亮
- 顶部有一个绿色主按钮：`新建日记`
- 左下角可以放一个便签纸样式的小鼓励语

导航菜单示例：
- 今日心情
- 情绪日记
- 情绪日历
- 数据分析
- 习惯追踪
- AI 心理助手
- 标签与回顾

标签区：
- 标题：`我的标签`
- 右侧有 `+`
- 标签用彩色小圆点 + 文本展示
- 标签颜色低饱和，例如蓝、红、紫、绿、橙

### 中间：主内容区
中间是核心区域，视觉上像一本打开的方格手帐本。

要求：
- 主内容区背景是白色或米白色方格纸
- 外层有类似笔记本封皮的浅棕色边框或阴影
- 左侧要有金属活页环/装订孔效果
- 页面四角圆润
- 纸张要有轻微阴影，不能是纯平面
- 内容像写在手帐纸上一样自然排布

顶部欢迎语：
- 居中显示类似胶带贴纸的标题
- 示例：`Hi, YJW ♡`
- 背后有一条浅米色胶带背景
- 下方有一句小标语：`每一次记录，都是心灵的温柔对话。`

### 右侧：数据卡片区
右侧宽度约 280px - 340px。
使用纵向卡片堆叠，每张卡片都是纸质卡片风格。

卡片内容包括：
- 情绪日历
- 本周情绪概览折线图
- 情绪分布环形图
- 今日一句话便签

卡片要求：
- 米白背景
- 圆角 12px - 18px
- 轻微阴影
- 边框极浅
- 不能太商业化，要像贴在手帐旁边的小纸片

## 3. 色彩规范

整体色调必须低饱和、温柔、自然。

主背景：
- 页面背景：`#F5EFE3` / `#F7F1E6` / `#F3E8D6`
- 纸张背景：`#FFFDF7` / `#FAF6EA`
- 方格线：`rgba(120, 100, 70, 0.08)`

主色：
- 治愈绿色：`#5D9B63`
- 浅绿色高亮：`#E8F2DF`
- 深绿色文字/按钮：`#3F7F4B`

辅助色：
- 暖黄色：`#F5C84C`
- 浅橙色：`#F6A04D`
- 柔红色：`#E96862`
- 淡紫色：`#A78BCB`
- 灰蓝色：`#8BA8C8`

文字颜色：
- 主文字：`#3C352B`
- 次级文字：`#7C7162`
- 弱提示文字：`#A79B8B`
- 不要使用纯黑

## 4. 字体风格

整体字体要像手写手帐，但必须保证可读性。

建议：
- 中文标题：使用手写感字体，例如 `Ma Shan Zheng`、`ZCOOL XiaoWei`、`LXGW WenKai`
- 正文：使用圆润、轻松的中文字体，例如 `LXGW WenKai`、`Noto Sans SC`、`Microsoft YaHei`
- 英文 Logo / 欢迎语：可以使用手写英文风格字体，例如 `Comic Sans MS`、`Patrick Hand`、`Caveat`

字体感觉：
- 不要使用过于正式的黑体后台风
- 不要使用科技感字体
- 字体间距稍微宽松
- 正文看起来像手帐记录，而不是表单输入

字号参考：
- Logo：24px - 28px
- 欢迎语：30px - 40px
- 卡片标题：16px - 18px
- 正文日记：17px - 20px
- 辅助说明：12px - 14px

## 5. 组件风格

### 按钮
主按钮：
- 背景绿色
- 白色文字
- 圆角 10px - 14px
- 高度 40px - 48px
- 有轻微阴影
- hover 时颜色稍微加深

按钮示例：
`新建日记`
`了解多建议 →`

次级按钮：
- 浅色背景
- 绿色文字
- 边框极浅
- 圆角柔和

### 卡片
所有卡片都要像纸片：
- 背景米白
- 圆角 14px - 20px
- box-shadow: 0 6px 20px rgba(80, 60, 30, 0.08)
- border: 1px solid rgba(120, 100, 70, 0.1)
- 卡片内部留白充足

### 输入区 / 日记编辑区
中间日记编辑区要像写在纸上：
- 方格纸背景
- 文字自然换行
- 不要像普通 textarea
- 工具栏是轻量的文字编辑工具栏
- 图标用线性图标，不要太现代商业感

日记内容示例：
```text
今天整体还算平静。
早上完成了课程设计的结构梳理，感觉思路清晰了很多，
有一种“慢慢来，也能做完”的踏实感。
下午和朋友聊了聊未来的计划，虽然有些迷茫，但彼此鼓励
让我安心了不少。
晚上散步听歌，风很温柔，心情也慢慢放松下来。
标签
标签样式：
• 胶囊形
• 浅色背景
• 低饱和色
• 文字颜色深一点
• 带关闭 x
• 示例：学习成长 ×、人际关系 ×、自我探索 ×
情绪选择器
情绪选择器是横向圆形表情按钮：
• 每个情绪是一个可爱的圆脸表情
• 选中状态有更大的浅绿色圆形背景
• 情绪包括：开心、平静、焦虑、难过、愤怒、疲惫
• 表情可以是手绘风，不要使用过于标准的 emoji
• 表情下方有小文字说明
6. 插画和装饰元素
必须加入少量手绘装饰，但不能喧宾夺主。
可用元素：
• 手绘太阳
• 绿色叶子
• 小爱心
• 小星星
• 胶带
• 回形针
• 便签纸
• 小植物
• 咖啡杯
• 书本
• 耳机
• 蜡烛
• 睡觉小猫
装饰原则：
• 每个页面只放少量
• 低饱和，线条柔和
• 不能变成儿童卡通风
• 要像手帐贴纸，不像游戏 UI
7. 图表风格
右侧图表必须融入手帐风格，不要用默认 ECharts 商务风。
折线图：
• 线条绿色
• 线条圆润
• 点是小圆点
• 网格线很淡
• 坐标轴文字小而柔和
环形图：
• 使用低饱和颜色
• 不要用鲜艳高对比色
• 图例简洁
• 数据展示要像“情绪分布贴纸”
日历：
• 每天用小圆脸表示情绪
• 当前日期用绿色圆圈强调
• 日期数字很小
• 整体像情绪贴纸月历
8. 质感要求
页面不能太干净、太像模板，要有真实纸张感。
需要实现：
• 背景有淡淡纸纹
• 中间笔记本区域有方格纸纹理
• 卡片有轻微阴影
• 便签有轻微倾斜感
• 胶带有半透明效果
• 装订环有一点立体感
• 但所有纹理都要很淡，不要脏乱
可以使用 CSS 实现：
• radial-gradient 模拟纸张噪点
• linear-gradient 模拟方格纸
• box-shadow 模拟纸片层次
• transform: rotate(-1deg) 模拟便签
• border-radius 做柔和纸张圆角
9. 交互风格
交互要轻柔、慢一点、治愈感。
hover：
• 卡片轻微上浮 2px
• 按钮颜色轻微变深
• 表情轻微放大
transition：
• 统一使用 0.2s - 0.3s ease
• 不要使用夸张动画
选中状态：
• 绿色浅底
• 柔和阴影
• 不要使用强烈边框
10. 不要出现的风格
请避免：
• SaaS 后台管理系统风
• 科技蓝紫渐变风
• 玻璃拟态风
• 高饱和霓虹色
• 纯白大面积无纹理背景
• 过硬的直角卡片
• 默认浏览器表单样式
• 太商业化的数据看板
• 太幼稚的儿童卡通风
• 过度复杂的插画
11. 页面复刻目标
最终页面应该让人第一眼感觉：
“这是一个温柔的情绪手帐网站，像一本真实的纸质日记本。用户每天可以在这里记录心情、贴标签、看情绪日历，也可以得到 AI 心理小建议。整体是米白纸张 + 绿色治愈系 + 手写字体 + 手帐贴纸 + 轻数据可视化的组合。”
请按照以上风格完成页面，不要自由发挥成其他现代网站风格。
